# Microgame

A Roblox project showcasing examples of a small free-to-play style Roblox game that is designed to be correct, separable, simple &, focused, and understandable.

### Example systems included are:

- Tutorial
- Repeatable game mechanic
- Soft currency
- Monetization
- Progression (Upgrades shop)
- Data saving

## Setup

- Run `git submodule update --init --recursive` to install dependencies
- Open `microgame.rbxp` in studio with `File > Open File`
